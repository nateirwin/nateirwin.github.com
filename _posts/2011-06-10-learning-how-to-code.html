As a "non-traditional" programmer -- I learned to write code on my own and studied Political Science in college -- I'll have to say that this article really hit home. Sure, Computer Science can be learned in a traditional environment, but there are so many resources available on the web today that I would find it difficult to justify paying for a degree to learn how to write code.
