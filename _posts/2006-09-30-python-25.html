The new version of Python, 2.5, has been released. It has been over a year and a half since the last release, but from looking at the "What's New" section of python.org, it looks like this release includes some major enhancements. These enhancements include the addition of many new modules to the core Python package. Multiple versions of Python can coexist on a computer, so you don't have to worry about conflicts with the package that comes installed with ArcGIS Desktop.
